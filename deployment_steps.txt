    1. EC2 access: It is virutal machine

    2. S3 bucket: To store artifact and model

    3. ECR: Elastic Container Registry; To save docker image in aws

    Description: About the deployment

    1. Build docker image of the source completed
    2. Push docker image to ECR
    3. Launch your EC2
    4. Pull your image from ECR to EC2
    5. Launch your docker image in EC2

    Policy:
    1. AmazonEC2ContainerRegistryFullAccess
    2. AmazonEC2FullAccess 
    3. AmazonS3FullAccess

        sudo sh get-docker.sh
